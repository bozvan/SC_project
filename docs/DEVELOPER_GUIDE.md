# 🔧 Руководство разработчика

## 🏗️ Архитектура проекта

### Модели данных (`models.py`)
```python
class Note:           # Заметки и закладки
class Task:           # Задачи  
class Tag:            # Теги
class WebBookmark:    # Веб-закладки
class Workspace:      # Рабочие пространства
```

### Менеджеры компонентов
- `DatabaseManager` - Единая точка работы с БД
- `NoteManager` - CRUD операции с заметками
- `TaskManager` - Управление задачами
- `BookmarkManager` - Работа с закладками
- `WorkspaceManager` - Управление пространствами
- `TagManager` - Работа с тегами

## 🗄️ Структура базы данных

### Основные таблицы
- `workspaces` - Рабочие пространства
- `notes` - Заметки и закладки
- `tasks` - Задачи
- `tags` - Теги
- `note_tag_relation` - Связи заметок с тегами
- `bookmark_tag_relation` - Связи закладок с тегами

## 🛠️ Разработка

### Настройка окружения
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск в режиме разработки
python main.py --debug
```

### Добавление нового функционала
1. Создайте модель в `models.py` (если требуется)
2. Реализуйте менеджер в соответствующем файле
3. Добавьте UI компоненты в `ui/`
4. Протестируйте функционал
