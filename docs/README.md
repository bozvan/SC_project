## 🚀 Краткое руководство

### Быстрый старт
1. Запустите `main.py`
2. Создайте новую заметку кнопкой "Новая заметка"
3. Начните печатать - автосохранение включено!

### Основные возможности
- 📝 Богатый текстовый редактор
- 🏷️ Система тегов
- ✅ Управление задачами
- 🔍 Умный поиск
- 🔔 Уведомления в системном трее
