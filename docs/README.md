# MINDSPACE 🧠

**Умный органайзер для управления заметками, задачами и закладками с поддержкой рабочих пространств.**

![Version](https://img.shields.io/badge/version-1.0.9-blue.svg)
![Python](https://img.shields.io/badge/python-3.8+-green.svg)
![PyQt6](https://img.shields.io/badge/PyQt6-6.0+-orange.svg)

## 🌟 Основные возможности

- **📝 Умные заметки** - Создание и организация заметок с поддержкой HTML/plain текста
- **✅ Менеджер задач** - Задачи с приоритетами, сроками и статусами выполнения  
- **🔖 Веб-закладки** - Автоматическое извлечение метаданных с веб-страниц
- **📁 Рабочие пространства** - Изоляция данных по проектам или темам
- **🏷️ Теги и категории** - Гибкая система тегирования для быстрого поиска
- **🎨 Темы оформления** - Темная и светлая темы с кастомизацией

## 🚀 Быстрый старт

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python main.py 
```

## 📁 Структура проекта

```
src/
├── assets/                  # Иконки / Картинки
├── core/                    # Ядро приложения
├── gui/                     # Пользовательский интерфейс
├── tests/                   # Файлы тестирования
├── ui/                      # Пользовательский интерфейс
├── widgets/                 # Виджеты
└── main.py                  # Точка входа
```

## 🔧 Требования

- Python 3.8+
- PyQt6
- SQLite3
- BeautifulSoup4
- Requests
