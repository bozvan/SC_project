# 📖 Руководство пользователя MINDSPACE

## 🎯 Основные функции

### Создание заметок
- Нажмите "Новая заметка" в интерфейсе
- Добавьте заголовок и содержимое
- Используйте HTML или обычный текст
- Добавьте теги для организации

### Управление задачами
```python
# Создание задачи с приоритетом
task_manager.create_task(
    note_id=1,
    description="Завершить проект",
    priority="high",
    due_date=datetime(2024, 12, 31)
)
```

### Работа с закладками
- Введите URL веб-страницы
- Система автоматически извлечет заголовок и описание
- Добавьте теги для категоризации

### Рабочие пространства
- Создавайте отдельные пространства для разных проектов и тем
- Переключайтесь между ними в интерфейсе
- Данные изолированы между пространствами

## 🔍 Поиск и фильтрация

### Текстовый поиск
- Ищите по заголовкам, содержимому и URL
- Поддержка частичного совпадения

### Поиск по тегам
- Фильтруйте по одному или нескольким тегам
- Комбинируйте с текстовым поиском

### Фильтрация задач
- По приоритету (высокий, средний, низкий)
- По статусу выполнения
- По сроку выполнения

## ⚙️ Настройки

### Темы оформления
- Темная тема (по умолчанию)
- Светлая тема
- Автоматическое сохранение выбора

### Рабочие пространства
- Создание новых пространств
- Переименование существующих
- Удаление (кроме пространства по умолчанию)

## 💡 Советы по использованию

1. **Используйте теги** - группируйте связанные элементы  
2. **Приоритеты задач** - отмечайте срочные задачи  
3. **Рабочие пространства** - разделяйте рабочие и личные проекты  
4. **Автоматические закладки** - экономьте время на добавление веб-страниц
